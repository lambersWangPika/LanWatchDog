## 三、问题与改进建议

### 🔴 高优先级

#### 1. 设备列表显示问题
| 问题 | 描述 |
|------|------|
| **设备名称为空** | 所有设备 `name` 字段为空，无法区分设备 |
| **设备类型识别不准确** | 2 台 VMware 虚拟机被识别为 `pc`，不够精确 |
| **无法自定义设备名** | 用户无法为设备设置友好名称 |

**建议**: 
- 添加设备名称编辑功能
- 支持设备备注/别名设置
- 显示首次发现时间帮助识别

#### 2. 流量监控数据显示问题
| 问题 | 描述 |
|------|------|
| **流量数据异常** | 显示大量公网 IP (如 72.145.35.110)，非局域网设备 |
| **数据无实际意义** | 所有设备显示相同的 `BytesIn: 1024, BytesOut: 512`，疑似模拟数据 |
| **单位显示不一致** | 首页显示 KB，流量页显示 GB |

**建议**:
- 过滤非局域网流量 (仅统计 192.168.x.x 段)
- 添加流量来源地域显示
- 统一单位显示

#### 3. 设备在线时长显示
| 问题 | 描述 |
|------|------|
| **在线时长全为 0** | `online_duration` 字段始终为 0 |

**建议**: 计算并显示设备在线时长
