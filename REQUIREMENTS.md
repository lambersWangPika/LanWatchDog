# 局域网设备监控系统 - 需求确认

## 项目概述

- **项目名称**: 局域网设备监控系统
- **平台**: Windows 桌面应用 (Go语言开发)
- **目标**: 实现局域网设备自动发现、实时监控、异常告警和日志记录

---

## 核心功能模块

### 1. 设备扫描模块
- [x] 定时扫描 (5/10/15/30分钟)
- [x] 手动扫描
- [x] IP范围配置 (CIDR)
- [x] Ping探测 + ARP表查询
- [x] MAC地址获取
- [x] 厂商识别
- [x] 设备类型识别

### 2. 设备监控模块
- [x] 在线状态实时显示
- [x] 最后在线时间
- [x] 在线时长统计
- [x] 设备白名单/黑名单
- [x] 设备备注/分组

### 3. 流量监控模块
- [x] 流量采集 (入站/出站)
- [x] 实时流量速率
- [x] 网卡选择
- [x] 全局/单独阈值设置
- [x] 流量预警 (实时告警)
- [x] 告警期间统计/总流量计算

### 4. 攻击检测模块
- [x] 暴力破解检测 (SSH/FTP/RDP)
- [x] 端口扫描检测
- [x] SYN/UDP/ICMP洪水检测
- [x] ARP欺骗检测
- [x] 异常连接检测
- [x] 攻击预警/溯源

### 5. 告警模块
- [x] 多级告警 (严重/高/中/低)
- [x] Windows气泡通知
- [x] 界面警告区
- [x] 声音提醒
- [x] 日志记录

### 6. 日志模块
- [x] 扫描/告警/操作/流量/攻击日志
- [x] JSON文件存储
- [x] SQLite数据库 (流量)
- [x] 日志搜索/导出/清理
